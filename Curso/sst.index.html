Curso de Shell (Bash) Scripting

1.¿Qué una shell?

Es una interfaz que permite a un usuario interactuar con el sistema operativo mediante comandos. Es decir, es un programa que proporciona una línea de comandos (CLI) a través de la cual los usuarios pueden ingresar comandos de texto para realizar diversas tareas.

Las funciones de una shell incluyen interpretar los comandos ingresados por el usuario, comunicarse con el núcleo del sistema operativo para ejecutar esos comandos y mostrar los resultados al usuario. La shell también puede proporcionar características como la autocompletación, el historial de comandos, y la capacidad de ejecutar scripts para automatizar secuencias de comandos. Esta última carácterística es la que convierte la shell en una herramienta imprescindible para el administrador de sistemas porque mejora la eficincia y garantiza la consistencia, la seguridad y escalabilidad de las operaciones. Existen diversas shells disponibles, y cada sistema operativo puede tener su propia shell predeterminada. Algunas de las shells más conocidas son Bash (Bourne Again SHell), PowerShell (en entornos Windows), Zsh (Z Shell), y Fish (Friendly Interactive SHell). En este curso nos ocuparemos sobre el scripting en sistemas Linux, y en particular, Bash.
